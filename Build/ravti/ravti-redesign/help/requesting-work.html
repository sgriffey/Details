Requesting Work

Today’s update introduces a highly requested feature: modules. Modules allow you to reuse code across many different projects and teams. Modules can include anything, from a set of basic functions that you like to use often, to extended Framer functionality, to a complete library for your company. They’re also great to keep your editable project small in size, and organize your code better.

Creating your own module is easy. Just create a new file in the modules folder and include it using the require keyword. Take a look at the example included in every new project, or read our full documentation. Modules are based on JavaScript and Node.js standards.

As a bonus example, we’ve reworked Facebook’s popular Framer Shortcuts by Cemre Güngör into a Framer module here.
