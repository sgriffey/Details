RTU Replacement Labor Scope

Scope for RTU-33 & RTU-34
  • Lock out, tag out existing high-voltage electrical service to packaged rooftop unit
  • Disconnect all low-voltage wiring (control wiring)
  • Provide and utilize crane and provide necessary rigging to remove existing packaged rooftop unit from existing location
  • Remove existing packaged rooftop unit from existing location, remove from premise, and legally dispose of unit
  • Utilize crane to fly and install new curb adapter
  • Utilize crane and provide necessary rigging to place new packaged rooftop unit in the same location the former unit resided
  • Properly seal and sit new packaged rooftop unit on curb adapter
  • Reconnect existing high-voltage electrical service to packaged rooftop unit, if new or additional high-voltage wiring is needed, provide and install this wiring
  • Reconnect back low-voltage wiring, if new or additional high-voltage wiring is needed, provide and install this wiring
  • New packaged rooftop unit should communicate with existing building thermostat
  • Complete project in accordance with all necessary city, county and/or state building code requirements (including but not limited to, hurricane tie-down straps)
  • Pull all necessary building permits from proper city, county or state code enforcement agency
  • The scope of work for being awarded for this project includes delivering a fully operating system, including any building automation system tie-in, where applicable
  • Install unit in accordance to manufacture’s IOM (installation and operation manual)
  • Provide authorized technician for start-up and first year warranty.
  • First year warranty to include all parts, all labor and any after hours emergency call(s) for the first year
