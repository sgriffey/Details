Tables

It's quite clear tables play an important role in the RAVTI product. Lets create lay some consistency in terms of layout styling and expectations.

Basic

Notes

Basic tables are minimal. They come with emphasized border-bottoms on the <thead> and <tr> element. To call a basic 3 column table (what's used in this example) give the <table> a @extend %full-table-across. Then apply @extend %col-3 on the <th> and <tr> elements.

Notes

Striped tables simply extend off the basic layout. To execute table stripes, @extend %striped-tr;, which will render an background-color: $off-white; on &:nth-child(odd) <tr> elements.
