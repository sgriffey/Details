Buttons

Notes

Buttons are styled by context. Buttons can be either neutral, encouraged or discouraged. Neutral buttons are stylized by @extend %button-neutral;, encouraged buttons as @extend %button-encourage;, and discouraged via @extend %button-discourage;. By adding the -large to any of button placeholders, the object will grow in size. Try to use @extend %button-encourage-large; @extend %button-discourage-large; @extend %button-neutral-large; only in simple views (e.g. app sign in) where the system is focused on a specific action.

Button Groups

Notes

Button groups will be used for places like page pagination and control toggles. The markup consists of a containing tag which has a @extend %button-group;, nested with tags which @extend %button-group-item;
